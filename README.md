## Design System

The project uses a custom design system based on Apple's design language:

- **Colors**: Clean, minimal palette with brand blue (#007AFF)
- **Typography**: System fonts with clear hierarchy
- **Spacing**: Consistent spacing scale (4px, 8px, 16px, 24px, 32px)
- **Radius**: Soft, rounded corners (16px cards, 28px buttons)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process required - it's ready to go!

## Customization

To customize the content, colors, or styling:

1. Edit `index.html` for content changes
2. Modify CSS variables in `styles.css` for design system changes
3. Update `script.js` for functionality changes
